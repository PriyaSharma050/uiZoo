A modern and interactive website showcasing a wide range of **UI elements** with live previews and copy-ready HTML/CSS code. Designed to help developers explore and use a variety of UI elements, including buttons, cards, forms, and loaders.

## Features

* Live interactive element previews
* One-click HTML/CSS copy
* Search and category filtering
* Smooth expandable previews
* Dark/Light mode

## Technologies

* **HTML5** (semantic structure)
* **CSS3** (Flexbox, animations, transitions, pseudo-elements)
* **JavaScript ES2024** (search, filter, expand, clipboard)
* **Icons8** for graphics and UI icons

## Purpose

To make it easy for developers to discover, understand, and reuse UI element patterns in their projects.
